# Reacts-webpack-cli

React项目构建Webpack脚手架

#### 安装
```
npm i -S reacts-webpack-cli
```
#### 命令集
一行命令搞定复杂的 `webpack` 配置问题，轻松构建基础版配置。
##### `npm init`
初始化 `webpack.config.js` 文件，提供一些基础属性设置，参见下面简化版 `webpack.config.js配置`。
##### `npm dev`
##### `npm build`
##### `npm ssr`
##### `npm rename`

#### `webpack.config.js` 配置
